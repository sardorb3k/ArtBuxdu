How Does it Work?

EnftyMart uses Authy as a 2fa provider. On enrollment in 2fa, we will send you a link to download authy and a code to confirm your phone number. You will then recieve 2fa requests through this app.

In cases where authy cannot be installed, sms messaging can be used for 2fa. This is a last case scenario as sms provides a less secure experience and will prevent you from enabling advanced 2fa settings within the site.
